PASOS CRUD FIRESTORE

	* Importar las dependencias de FIRESTORE (La nueva versión de firebase incorpora nuevas dependencias para Firestore)-->
		* import { provideFirestore, getFirestore } from '@angular/fire/firestore';
		* import { FIREBASE_OPTIONS } from '@angular/fire/compat';
		* import {initializeApp, provideFirebaseApp} from '@angular/fire/app';
	* Añadir credenciales del proyecto de firebase en el app.module.ts
	* Inyectar las importaciones en la propiedad imports: 
		* provideFirestore(() => getFirestore())
		* provideFirebaseApp( () => initializeApp(firebaseConfig))
	* Inyectar servicio Firebase en providers:
		* { provide: FIREBASE_OPTIONS, useValue: firebaseConfig }
